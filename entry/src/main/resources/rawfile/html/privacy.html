关于 Voot 与隐私的声明

Voot（以下简称“本应用”）由 杨子啸（以下简称“我们”）向您提供，是一款在 HarmonyOS 上运行、基于用户自有大型语言模型 API 的同声传译与文本翻译工具。本隐私声明由我们为处理您的个人信息而制定。

我们非常重视您的个人信息和隐私保护，将会按照适用的法律法规以及业界通行的安全标准，为您的个人信息提供相应的安全保护措施。

1. 我们如何收集和使用您的个人信息

我们仅在具有合法性基础的情形下处理您的个人信息。根据适用法律，我们可能会基于您的同意、为履行/订立您与我们的合同所必需、履行法定义务所必需等合法性基础，收集和使用您的个人信息。

1.1 为您提供同声传译与文本翻译功能

在您使用本应用进行录音、同声传译或文本翻译时，为实现核心功能、履行我们对您的服务承诺，我们会在本地处理下列信息：

  • 通过麦克风采集的语音数据（包括应用在前台或后台运行时）；
  • 由本地语音识别模块生成的文本内容；
  • 您选择的源语言、目标语言、翻译引擎等配置；
  • 翻译结果文本；
  • 通过系统语音合成能力生成的朗读音频。

处理方式说明：

  • 录制的音频仅在您的设备本地用于识别和翻译处理，不会上传至我们的服务器或其他不必要的第三方；
  • 翻译完成后，音频数据会在本地处理流程结束后被删除；
  • 当您选择使用某一第三方 LLM/翻译 API 时，本应用仅会在您的授权下，将经本地处理后的文本内容通过网络发送至您配置的 API 服务提供者，以获取翻译或生成结果。该等第三方可能会独立收集和处理您的相关数据，请您务必仔细阅读并遵守相应第三方的隐私政策；
  • 当您开启实时朗读功能时，翻译结果文本将调用系统语音合成服务进行朗读，该过程在本地或系统层面完成，遵循 HarmonyOS 系统相关隐私规范。

若您拒绝提供上述信息，可能导致无法正常使用本应用的同声传译和翻译功能。

1.2 历史记录与收藏功能

为便于您回顾和管理翻译内容，本应用提供历史记录和收藏功能：

  • 每次语音识别会话的原文和译文将自动保存至本地数据库；
  • 您可以收藏重要的翻译记录以便日后查阅；
  • 历史记录包括会话时间、原文内容、翻译结果等信息。

处理方式说明：

  • 所有历史记录和收藏数据均存储在您设备本地的加密沙箱中，不会上传至我们的服务器；
  • 您可以随时在应用内查看、删除单条或全部历史记录；
  • 卸载本应用将同时删除所有本地存储的历史记录和收藏数据。

1.3 文档扫描与图片保存功能

本应用提供文档扫描功能，帮助您快速识别和翻译纸质文档或图片中的文字：

  • 通过相机扫描文档，生成可用于文字识别的图片；
  • 扫描的图片与识别、翻译结果一并保存至扫描历史记录；
  • 您可以在预览界面放大查看扫描图片，并选择保存到系统相册。

处理方式说明：

  • 扫描功能调用 HarmonyOS 系统提供的文档扫描控件（Vision Kit），扫描过程由系统控制；
  • 在扫描过程中，系统可能会使用光照传感器检测环境亮度，用于暗光下打开手电筒以提供更清晰的文档图片，该信息仅在端侧使用；
  • 扫描生成的图片仅保存在您设备本地的应用沙箱中，不会上传至我们的服务器或华为云端，系统不留存扫描的图片数据；
  • 当您选择保存图片到相册时，本应用会通过系统弹窗请求您的授权确认，图片将保存到您的系统媒体库；
  • 您可以随时在应用内删除扫描历史记录，包括关联的扫描图片；
  • 卸载本应用将同时删除所有本地沙箱中的扫描图片，但已保存到系统相册的图片不受影响。

华为 Vision Kit 个人数据处理说明：本应用使用的文档扫描控件由 HarmonyOS Vision Kit 提供。根据华为官方 Vision Kit 个人数据处理说明，在使用文档扫描功能的选图扫描和相机扫描过程中，Vision Kit 会读取或拍摄待扫描的图片用于文档扫描处理，同时使用光照传感器用于暗光下打开手电筒。上述个人信息均在端侧使用，华为云端不存储用户数据，图片数据不留存。

1.4 跨设备流转功能

为提升多设备协作体验，本应用支持在您的多台 HarmonyOS 设备（如手机与平板）之间进行任务流转：

  • 当前的翻译会话状态（包括原文、译文、API 配置等）；
  • 字幕样式设置（字体大小、颜色、背景色等）；
  • 录音状态和画中画窗口状态。

处理方式说明：

  • 流转功能需要您的设备登录同一华为账号并处于同一网络环境；
  • 流转数据通过 HarmonyOS 系统提供的分布式能力在设备间直接传输，遵循系统安全机制；
  • 流转过程中传输的数据不经过我们的服务器，由 HarmonyOS 系统负责加密传输；
  • 您需要授予分布式数据同步权限才能使用此功能。

1.5 保护您的 API 密钥与账户安全

为了帮助您在本应用中安全地使用自有模型 API，我们会在本地处理以下信息：

  • 您主动输入和保存的各类第三方服务 API Key、访问令牌或接口配置；
  • 用于解锁、读取上述密钥的人脸/生物识别验证结果（由 HarmonyOS 提供）。

处理方式说明：

  • 您的 API 密钥仅保存在本应用的本地沙箱中，并采用系统提供的安全机制进行加密存储，不会上传至我们的服务器；
  • 当您发起调用请求时，本应用仅在本地解密并使用该密钥与您选定的第三方服务通信；
  • 人脸/生物识别的比对过程由操作系统安全模块完成，我们不保存您的面部图像、特征模板等生物识别原始数据，仅使用验证结果（通过/未通过）来控制对本地密钥的访问。

若您不希望使用人脸/生物识别保护密钥，可在系统关闭相关选项，但可能影响密钥访问的便捷性与安全性。

1.6 保障应用基础功能与安全运行

为保证本应用稳定、安全地运行，我们可能会在本地处理有限的设备及运行信息，例如：

  • 应用版本号、系统版本、设备型号；
  • 应用崩溃信息、错误日志、性能统计等运行状态。

上述信息主要用于问题排查和性能优化，通常仅在本地分析处理。如需在您主动反馈问题时提交相关日志，我们会在您明确同意的前提下，通过您选择的方式（例如邮件、开发者工具等）与我们共享。

1.7 第三方服务

本应用支持您接入第三方服务（例如：OpenAI、DeepL、Ollama、豆包等），相关服务由对应服务提供者向您单独提供。

在您配置并使用这些第三方服务时：

  • 本应用会按您的指示，将必要的请求数据（如文本内容、目标语言、API Key 等）发送至相应的第三方服务器；
  • 该等第三方将根据其自身隐私政策对您的个人信息进行处理，我们强烈建议您在使用任何第三方服务前，仔细阅读并理解其隐私政策与服务条款。我们对第三方服务的数据处理行为不承担责任。
  • 您应当自行了解并同意相关第三方的条款与隐私政策后再决定是否使用。

1.8 实名认证服务

为满足中国大陆地区相关法律法规对互联网信息服务的实名制管理要求，当您在中国大陆地区使用本应用时，我们需要对您进行实名认证。

本应用不提供账号注册或登录功能，实名认证仅用于合规目的。我们通过采购第三方服务来实现手机号验证：

  • 手机号认证服务：由阿里云计算有限公司提供，用于验证您的手机号码真实性；
  • 短信验证码服务：由上海时巴克科技有限公司提供，用于向您的手机发送验证码。

处理方式说明：

  • 您输入的手机号码将通过加密通道发送至上述服务提供商进行验证；
  • 验证通过后，我们仅在本地记录您的认证状态（已认证），仅会存储您的手机号码；
  • 上述服务商可能会根据其隐私政策处理您的手机号码等信息，请您在使用前仔细阅读相关服务协议。

2. 设备权限调用

为实现上述功能并保障应用正常运行，本应用可能会在使用过程中向您申请以下系统权限。我们仅会在您主动授权后使用对应权限，并严格遵循最小必要原则：

  • 麦克风权限：用于录制语音，以实现同声传译与语音输入翻译功能。
  • 相机权限：用于文档扫描功能，通过相机拍摄文档或图片以进行文字识别和翻译。
  • 媒体库写入权限：用于将扫描的图片保存到系统相册，需要通过系统弹窗获得您的授权确认后方可保存。
  • 网络访问权限：用于将本地生成的文本内容发送至您配置的第三方 API 服务，并接收翻译或生成结果。
  • 生物识别/人脸识别权限：由 HarmonyOS 提供的安全能力，用于在本地验证您的身份，以解锁和使用保存在沙箱中的 API 密钥。我们不存储您的面部图像或生物特征原始数据。
  • 本地存储权限（如适用）：用于在本地保存您的翻译配置、API 连接设置、历史记录、收藏数据以及必要的缓存数据。
  • 后台运行权限：用于在应用切换至后台或锁屏状态下，持续进行录音、翻译与朗读，确保同声传译体验不中断。
  • 分布式数据同步权限：用于在您的多台 HarmonyOS 设备之间进行任务流转，同步翻译会话状态、字幕样式等信息。此权限仅在您主动发起跨设备流转时使用。

您可以在设备系统设置中随时管理或撤回上述权限授权。但在您撤回相关权限后，某些功能可能无法使用或无法达到预期效果。

4. 信息存储地点及期限

4.1 存储地点

  • 我们本身不运行面向用户的集中式业务服务器，您的翻译内容、录音数据和 API 密钥主要存储并处理于您的本地设备沙箱中；
  • 当您调用第三方 API 时，相关请求数据会直接从您的设备发送至对应服务提供者的服务器，该等服务器可能位于中国境内或境外，具体以第三方服务提供者的说明为准。

4.2 存储期限

  • 录制音频仅在本地完成识别与翻译处理所必需的时间内短暂保留，之后将被删除；
  • 翻译文本、应用配置和 API 密钥会在您使用本应用期间保留于本地设备，直至您删除相关数据或卸载本应用；
  • 运行日志等技术数据的本地保存期限，将根据问题排查和性能优化的需要合理设置，超期后将删除或进行匿名化处理。
